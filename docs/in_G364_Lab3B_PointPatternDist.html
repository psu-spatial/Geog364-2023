Welcome to Lab 3B on Point Pattern Analysis. A point pattern dataset is comprised of events or objects occurring in a defined study region. These events could represent anything with a specific measurable point location.

Objective: The goal of Lab 3B is to learn some introductory spatial statistical approaches for characterizing the spatial properties of a point pattern.

  • This week: We will be focusing on point pattern DISTANCE analysis e.g. nearest neighbour
